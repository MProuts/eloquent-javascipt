# Eloquent Javascipt - Notes & Exercises

A little place to scribble notes and exercises as I work through
"Eloquent Javascript" by Marijn Haverbeke

Directory structure
---
```
├── chapter_00/
│   ├── exercises/
│   ├── notes.md
│   └── test.js
```
To run tests
---
```sh
npm test chapter_02/test.js
```

To start a REPL
---
```sh
npx babel-node
```
